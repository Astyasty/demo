A variety of predefined service options are available to product clients through plans.

You can create and customize an unlimited number of plans to meet your needs. These plans reflect standard user attributes and fully align with the state flow.

With the latest system update, user records now include additional properties, as described in the latest API reference. For instance, users can transition to a 'Signing' state before confirming a plan purchase.

Note. After the feature launches, the system update may require adjustments to customer service processes.
