Creating a partition on a disk in Linux

Creating a partition on a disk helps to organize different memory regions for specific uses. It also helps to install multiple operating systems on one machine and minimize the damage in case of disk corruption.

All commands must be run as the root user.

To create a partition in a disk in Linux:

  1. Run the command to list all existing disks and partitions: sudo fdisk -l
  2. Run the command to list all existing disks and partitions: sudo fdisk
  3. Press n to create a new partition.
  4. Select one of the following steps to choose the partition type:
    • To create a primary partition, press p.
    • To create an extended partition, press e.
  5. Enter the number of a partition.
  6. Enter the first sector number.
  7. Enter the last sector number or number of sectors or a partition size.
  8. Press w to write the new partition on the disk.

To view a new partition table, press p. To change a partition type, press t and enter a number for a partition type.
